# 2235. Add Two Integers

**Difficulty:** Easy

## Problem Description
Given two integers `num1` and `num2`, return their sum.

## Examples

### Example 1
```
Input: num1 = 12, num2 = 5
Output: 17
```

### Example 2
```
Input: num1 = -10, num2 = 4
Output: -6
```

## Constraints
- `-100 <= num1, num2 <= 100`

## Approach
Direct addition and return.

## Time & Space Complexity
- **Time Complexity:** `O(1)`
- **Space Complexity:** `O(1)`
